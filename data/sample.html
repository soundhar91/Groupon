New Blocking Controls


Blocking controls for each detected tracker are now available by default. The following screens will walk you through using (the global blocking toggle) and (the site-specific unblock button).

Global Unblock


Ghostery will not block a tracker if the global blocking toggle is blue and positioned to the left.

Global Block


Ghostery will block this tracker on all sites if the global blocking toggle is red and positioned to the right.

Site-specific Unblock


Ghostery will not block this tracker on the current site if you press the site-specific unblock button, even if the tracker is globally blocked.

Pause Blocking


Ghostery will pause and temporarily stop all blocking if you press Pause Blocking below. The global blocking toggle will turn yellow to indicate this, regardless of position.

Whitelist Site


Ghostery will not block any trackers on the current site if you press Whitelist Site below. The global blocking toggle will turn green to indicate this, regardless of position.
